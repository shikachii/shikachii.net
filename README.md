# 動かし方
## npmを用いる場合
```
# Recommend Node.js version 10.16.0 or higher.
$ npm install
$ npm run serve
```

## docker composeを用いる場合
```
$ docker-compose up --build
```

## 閲覧
`localhost:8080`にアクセスすることで確認可能。
